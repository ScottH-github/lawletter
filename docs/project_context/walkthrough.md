# LegalLetter AI - 專案導覽

恭喜！您的 **LegalLetter AI** (存證信函產生器) 已經建置完成。本專案旨在協助使用者產生符合中華郵政格式的專業存證信函。

## ✅ 已完成功能

### 1. 專案初始化與架構
- **Next.js 14 App Router**: 使用最新的 Next.js 架構。
- **Tailwind CSS & Shadcn/UI**: 建立乾淨、專業的「法律事務所」風格介面 (深藍/白/岩灰)。
- **字型優化**: 整合 **Noto Sans TC** (Google Fonts 與本地檔案)，確保介面與 PDF 皆無亂碼。

### 2. 核心功能：輸入表單
- **多步驟表單**: 
    - **步驟 1**: 填寫寄件人與收件人資訊。
    - **步驟 2**: 輸入案情描述 (白話文)。
    - **步驟 3**: 預覽與生成。
- **狀態管理**: 使用 **Zustand** 管理複雜的信函狀態。

### 3. AI 律師整合
- **角色扮演**: AI 被設定為「台灣律師」，能將口語轉化為專業法律用語 (如：台端、本人、茲、限期)。
- **語氣調整**: 支援「專業客觀」、「語氣強硬」、「語氣溫和」三種模式。
- **API Route**: 實作 `/api/rewrite`，若無 API Key 會自動提供高品質的模擬回應供測試。

### 4. PDF 生成 (中華郵政格式)
- **嚴格網格排版**: 使用 `@react-pdf/renderer` 繪製符合郵局標準的網格 (通常為 20字/行)。
- **自動分頁**: 內容過長時自動延伸至下一頁。
- **正式格式下載**: 點擊按鈕即可下載生成的 PDF 檔案。

## 🚀 如何執行

1. **啟動開發伺服器**:
   ```bash
   npm run dev
   ```
2. **開啟瀏覽器**: 前往 `http://localhost:3000`。
3. **測試流程**:
   - 填寫基本資料。
   - 輸入案情 (例如：「房客欠租」)。
   - 點擊「AI 生成信函」(若無 Key 會看到模擬結果)。
   - 在最後一步點擊「下載正式 PDF」。

## ⚠️ 注意事項

- **OpenAI API Key**: 若要在正式環境使用真實 AI 功能，請在 `.env` 中設定 `OPENAI_API_KEY`。
- **PDF 字型**: 當前設定使用專案內的 `public/fonts/NotoSansTC-Regular.otf`，確保 PDF 生成時中文字元正確顯示。
